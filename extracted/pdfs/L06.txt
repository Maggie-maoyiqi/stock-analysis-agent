6.1 Variable Selection
Consider the general regression model (full model with k independent variables)
y= β0 + β1 x1 +···+ βk xk + ε.
In some studies, it is not necessarily that all of the k independent variables are useful to
predict Y . One main objective of statistical analysis is to obtain a parsimonious statistical
model so that interpretations of the model can be made simple.
Aim of this section is to introduce some statistical methodologies or techniques to reduce
the number of independent variables so that the final model is reasonably simple and hope-
fully, easy to make appropriate interpretations.

6.1.3 Sequential methods
When the number of independent variables k is very large, evaluating all (2k
−1) possible sub-
sets regression models may not be feasible. The idea of sequential method is to sequentially
include and/or drop one variable at a time.
1. Forward selection method:
(F1) (F2) (F3) Starting with the model with no independent variable (intercept only).
Add the MOST SIGNIFICANT variable into the model based on a criterion (e.g. if
the F statistic is greater than the entry level, e.g. 4; or if the p -value is smaller than
the entry value, e.g. 0.05).
Repeat (F2) until no non-selected variable can be added by satisfying the criterion.
2. Backward elimination method:
(B1) Starting with the full model (include all k independent variables and the inter-
cept).
(B2) Remove the MOST INSIGNIFICANT variable in the model based on a criterion (e.g.
if the F statistic is smaller than the removal level, e.g. 4; or if the p -value is greater
than the removal level, e.g. 0.05).
(B3) Repeat (B2) until no variable in the model can be eliminated by the criterion in
(B2).
3. Stepwise method:
(F1) (F2) (B2) (S3) Start with model with no independent variable (intercept only).
Add the MOST SIGNIFICANT variable into the model based on a criterion (e.g. if
the F statistics is greater than the entry level, e.g. 4; or if the p -value is smaller than
the entry value, e.g. 0.05).
Remove the MOST INSIGNIFICANT variable in the model based on a criterion (e.g.
if the F statistics is smaller than the removal level, e.g. 4; or if the p -value is greater
than the removal level, e.g. 0.05). Note that the entry level should never be smaller
than the removal level; otherwise cycling is possible.
Repeat (F2) and (B2) until no non-selected variable can be added and no variable
in the model can be eliminated.
Changing the criteria in (F2) and (B2) may result in a different final model.
